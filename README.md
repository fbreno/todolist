## Instalação

1. Clone este repositório:
2. Navegue até o diretório do projeto:
3. Instale as dependências:

   Com npm:

    ```bash
    npm install
    ```

## Rodando a Aplicação

Para rodar a aplicação em modo de desenvolvimento, utilize o comando:

Com npm:

```bash
npm run serve
```
